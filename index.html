FruitWater.css

FruitWater.css is a simple, class-less css micro-framework.

It is made for fun, to enjoy myself, while trying to push as far as possible, what can be done using no css classes or ids.

It is based and inspired of a few projects including:

  • Water.css - inspired both by name as well as many of the components. In fact this started as a fork of it
  • Skeleton css - for some of the design and ideas inspiration
  • MVP.css - gave me the inspiration for the <aside> columns system
  • Tailwind css & Tailwind UI - inspired by their beautiful designs

Why and what for?

  • For fun
  • To see how far could I take a class-less system, while implementing some very opinionated but generic ideas
  • Good-enough for quick prototype, especially on Codesandbox or something similar
  • Small size (< 2kb)
  • Beautiful
  • No classes

Columns

Using the aside element you can easily set elements side-by-side in an equal-width format, just like a column/grid type system.

Column 3

        </aside>
